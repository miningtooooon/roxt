# Telegram Mini App (Rewards UI)

This is a **demo UI** with the neon/glass style.

## Run locally
1) Copy `.env.example` to `.env` and set `VITE_API_URL`.
2) Install & start:
```bash
npm install
npm run dev
```

## Telegram integration note
For production, replace the demo login with Telegram WebApp `initData` verification on the backend.
